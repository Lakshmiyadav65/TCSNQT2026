Q1: DISCOUNT CALCULATOR

Problem

Given a purchase amount, apply discount: (amount < 1000: 5% | < 5000: 10% | else: 15%). Print final amount with 2 decimal places.

🔴 EXPLANATION BY LAKSHMI BUILDS ❤️

Learn conditional branching! The goal is to catch the right "bucket" based on spendings and format the output like currency.

OFFICIAL SOLUTIONS
amount = float(input())
rate = 0.05 if amount < 1000 else (0.10 if amount < 5000 else 0.15)
print(f"{amount * (1 - rate):.2f}")
import java.util.*;
public class DiscountCalculator {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        double d = sc.nextDouble();
        double r = (d < 1000) ? 0.05 : (d < 5000 ? 0.10 : 0.15);
        System.out.printf("%.2f%n", d * (1 - r));
    }
}

Q3: SOLDIER ARRANGEMENT

Problem

Count valid arrangements of N soldiers (values 1 to R). first=1, last=end, neighbors differ.

🔴 EXPLANATION BY LAKSHMI BUILDS ❤️

We use Recursion with Memoization (caching) to efficiently count legal sequences without redundant counting.

OFFICIAL SOLUTIONS
from functools import lru_cache
n, r, end = map(int, input().split())
@lru_cache(None)
def count(pos, last):
    if pos == n - 1: return 1 if last == end else 0
    res = 0
    for v in range(1, r + 1):
        if (pos == 0 and v == 1) or (pos > 0 and v != last): res += count(pos + 1, v)
    return res
print(count(0, -1))
import java.util.*;
public class SoldierArrangement {
    static int n, r, end;
    static Map memo = new HashMap<>();
    static int count(int p, int l) {
        if (p == n-1) return (l == end) ? 1 : 0;
        String key = p + "," + l;
        if (memo.containsKey(key)) return memo.get(key);
        int res = 0;
        for (int i=1; i<=r; i++) if ((p==0 && i==1) || (p>0 && i!=l)) res += count(p+1, i);
        memo.put(key, res); return res;
    }
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        n = sc.nextInt(); r = sc.nextInt(); end = sc.nextInt();
        System.out.println(count(0, -1));
    }
}

Problem

🔴 EXPLANATION BY LAKSHMI BUILDS ❤️

Crucial logic: apply rates step-by-step for each tier, don't use a single rate for the whole duration!

OFFICIAL SOLUTIONS
h = int(input())
if h <= 2: c = h * 100
elif h <= 5: c = 200 + (h-2)*50
else: c = 200 + 150 + (h-5)*20
print(c)
import java.util.*;
